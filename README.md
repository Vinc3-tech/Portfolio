# Vincenzo Venturino – Portfolio Website

Portfolio personale sviluppato per presentare il mio percorso come sviluppatore web, i progetti realizzati e le competenze tecniche acquisite.

Il sito è stato progettato con particolare attenzione all'esperienza visiva, all'identità grafica e alle animazioni, utilizzando tecnologie moderne per creare un'interfaccia coinvolgente e responsive.

## Descrizione

Questo progetto nasce con l'obiettivo di rappresentare la mia identità come sviluppatore attraverso un'esperienza web moderna e immersiva.

Il portfolio include:

- Presentazione personale
- Sezione dedicata ai progetti
- Contatti
- Animazioni avanzate con GSAP
- Layout responsive per desktop e mobile

---

## Tecnologie Utilizzate

### Frontend

- HTML5
- CSS3
- JavaScript

### Librerie

- GSAP (GreenSock Animation Platform)
- ScrollTrigger
- SplitText

### Design

- Responsive Design
- Mobile First Principles
- Custom Typography
- Smooth Animations
